Image Classification

Image Classification is a technique which can classify an image into several categories. The Azure model used on this page is trained using specialised data sets such as images made up of landmarks and labels. Other domain specific applications can be trained, or applied for - such as the case with Azure's Celebrity domain models. Read more here.
