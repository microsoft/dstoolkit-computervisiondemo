Video + Motion Detecton

Every Computer Vision technique that has been demonstrated on this website (and more) can be applied to videos on a frame by frame basis. For an example using a video of cars on a motorway, we can apply simple motion detection techniques to detect the motion in the video. Once this is over a threshold that we define we can activate an 'alarm' signaling there is motion near the camera or an object of a certain size has moved passed. Please see the top left corner for an indication when alarm is activated.

You can imagine the applicability of techniques such as these:- security cameras, manufacturing checks (on/ off), detecting wildlife of a certain size, ect. Read more about Azure Computer Vision here.
